# Poker Sevens Strategy Assistant 🃏

這是一個專為撲克牌遊戲「排七」設計的戰略輔助工具。它的核心目標不是讓你拿第一名，而是透過 **「斷路 (Blocking)」** 與 **「點數管理」** 確保你絕對不會成為最後一名輸家。

## ✨ 功能亮點
- **即時出牌建議**：根據桌面局勢，分析最優出牌路徑。
- **斷路器監控**：標記出你手中能卡死對手的關鍵牌（5, 6, 8, 9）。
- **避坑指南**：當沒牌可出時，自動計算點數最小的蓋牌方案。

## 🚀 如何執行
1. 安裝環境：`pip install -r requirements.txt`
2. 啟動程式：`streamlit run app.py`
